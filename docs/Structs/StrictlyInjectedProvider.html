StrictlyInjectedProvider

public struct StrictlyInjectedProvider<I> : InjectedProvider where I : Injector

The InjectedProvider for StrictInjector. All factories will be evaluated immediately. If the factory throws an error, it will be stored and throwed on StrictlyInjectedProvider.resolve(withInjector:).
